# ai-artifacts Documentation Context

This repository has two documentation areas:

- Root documentation such as `README.md` for repository/product orientation.
- `docs/whitepaper/` for whitepaper sources and PDF generation tooling.

For whitepaper edits, preserve the evidence-first editorial rule: a table summarizes an argument already explained; it must not replace proof.

For package documentation, avoid Travel Storefront-only assumptions unless they are clearly marked as examples.
